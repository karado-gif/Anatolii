# Karado

Мобильное приложение для покупки автомобилей с японских аукционов и отслеживания их доставки.

## Описание
Karado — это кроссплатформенное приложение (iOS/Android) для частных лиц, позволяющее искать, просматривать и покупать автомобили напрямую с японских аукционов, а также отслеживать их доставку до Владивостока.

---

## Основной функционал
- **Поиск авто:**
  - Фильтры: страна (Япония), номер кузова, поколение, цвет, бюджет, объем двигателя, тип топлива
- **Список машин:**
  - Просмотр результатов поиска
- **Карточка авто:**
  - Фото, аукционный лист, история торгов, цена, рекомендованная ставка, описание состояния, опции
- **Мультиязычность:**
  - Русский и английский языки
- **Мок-данные:**
  - Для демонстрации работы приложения используются тестовые данные

## Установка и запуск
1. **Установите Node.js** (если не установлен): https://nodejs.org/
2. **Установите Expo CLI:**
   ```sh
   npm install -g expo-cli
   ```
3. **Установите зависимости проекта:**
   ```sh
   npm install
   ```
4. **Запустите проект:**
   ```sh
   npx expo start
   ```
5. **Откройте приложение:**
   - На эмуляторе iOS/Android через Expo DevTools
   - Или на телефоне через приложение [Expo Go](https://expo.dev/client)
